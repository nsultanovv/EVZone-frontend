# BuyEV - Electric Vehicle Platform

A modern, multi-page React site for the EV auto niche. Built with React, Tailwind CSS, and Lucide React.

## Tech Stack

- **React 18** with hooks
- **Tailwind CSS** (core utility classes)
- **Lucide React** for icons
- **Vite** for build tooling

## Features

- **6 Pages**: Home, How It Works, Testimonials, Pricing, FAQ, Contact
- **Design**: Teal/cyan gradient scheme, glassmorphism, dark theme (#08080f)
- **Animations**: fadeInUp hero, IntersectionObserver scroll reveals, animated counters, floating blobs
- **Responsive**: Mobile-first with breakpoints for tablet and desktop

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:5174](http://localhost:5174) in your browser.

## Build

```bash
npm run build
npm run preview
```
